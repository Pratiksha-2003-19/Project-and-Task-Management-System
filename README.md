# 📌 Team Task Manager

A full-stack **Task Management System** built using **Spring Boot, Spring Security (JWT), PostgreSQL, and React**.  
It helps teams manage projects, assign tasks, track progress, and improve productivity.

---

## 🚀 Live Overview

This application provides:
- User Authentication (JWT-based)
- Role-based Access Control (Admin / Member)
- Project Management
- Task Assignment & Tracking
- Secure REST APIs

---

## 🏗️ Tech Stack

### Backend
- Java 17
- Spring Boot 3.2.5
- Spring Security (JWT)
- Spring Data JPA
- Hibernate
- PostgreSQL (NeonDB / Railway)
- Maven

### Frontend (if applicable)
- React.js
- Axios
- React Router

---

## 🔐 Features

### 👤 Authentication
- User Registration
- Login with JWT Token
- Secure API Access

### 👥 Roles
- ADMIN: Full access (projects + tasks + users)
- MEMBER: Assigned tasks only

### 📁 Project Management
- Create Project
- Assign Members
- Track Completion Status

### ✅ Task Management
- Create Tasks
- Assign Tasks to Users
- Priority Levels (LOW / MEDIUM / HIGH)
- Status Tracking (TODO / IN_PROGRESS / COMPLETED)
